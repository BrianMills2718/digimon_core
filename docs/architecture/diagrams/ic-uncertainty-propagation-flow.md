## Key Propagation Principles

1. **Independence Assumption**: Default to RSS unless dependencies are known
2. **Conservative Estimation**: When uncertain, assign higher uncertainty
3. **Transparent Calculation**: All math is hard-coded, not LLM-generated
4. **IC Standards**: Output maps to standardized IC probability bands
